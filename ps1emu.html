PS1 Emulator

What is it?

The project is a PS1 emulator made using Vulkan as a renderer.
Currently, it can run about 60 different instructions and can handle vertex color primitives.

Where can I access the source code?

As this is still in development, the source code is not currently available. However, once finished the source will be up on my github.
